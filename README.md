# Automated Ammonia Detection and Neutralizing System in Poultry Farms

The **Automated Ammonia Detection and Neutralizing System in Poultry Farms** is an IoT-based smart farming solution designed to monitor and control harmful ammonia gas levels inside poultry sheds. Ammonia generated from poultry waste can cause respiratory diseases, reduced growth, lower egg production, and increased mortality among birds. To overcome these challenges, the system uses an **MQ-135 ammonia sensor** to continuously detect ammonia concentration, while a **DHT11 sensor** monitors temperature and humidity. An **ESP32 microcontroller** processes the sensor data and compares the ammonia level with predefined safety thresholds. When the concentration exceeds the safe limit, the controller automatically activates a **relay-controlled DC pump** that sprays a neutralizing solution, such as a citric acid mist, to reduce ammonia levels. Simultaneously, the system displays real-time readings on a **16×2 LCD**, sends updates to an **IoT dashboard (ThingSpeak)**, and provides alerts to notify the user. Once the ammonia concentration returns to a safe range, the spraying mechanism is automatically turned off, ensuring efficient use of water and energy. This automated system reduces manual monitoring, improves air quality, enhances poultry health and productivity, and provides a cost-effective, scalable solution suitable for both small- and large-scale poultry farms. Future enhancements include AI-based predictive analytics, solar-powered operation, support for multiple gas sensors, and mobile application integration for smarter and more sustainable poultry farming.
